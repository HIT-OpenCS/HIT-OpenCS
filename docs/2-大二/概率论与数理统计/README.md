
# 概率论与数理统计

## 简介

平时作业占考核成绩的20%; 课程论文、课堂表现占考核成绩的10%; 期末试卷占考核成绩的70%。

## 关于考试
期末试卷，占考核成绩的70% ，即70分，包括：

* 5道填空题占15分
* 5道选择题占15分
* 5道计算证明题占40分
